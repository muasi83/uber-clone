SECTION 2: 🛣️ THE BIG JOURNEY — How Each Ride Works Step by Step

┌───────────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                           │
│  👤 RIDER PICKS UP PHONE → Opens App → Sees Map → Types "Where to?"                      │
│                                                                                           │
│       ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ● ●               │
│       ●                                                                 ●               │
│       ●  ①  📲 Rider picks pickup & dropoff on the map                  ●               │
│       ●      ↓                                                          ●               │
│       ●  ②  🖥️ App asks server: "What's the price?"                    ●               │
│       ●      ↓                                                          ●               │
│       ●  ③  🌐 Google Maps responds: "Distance = 5 km"                 ●               │
│       ●      ↓                                                          ●               │
│       ●  ④  🖥️ Server calculates: "Price = 25 SAR"                    ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑤  📲 Rider sees price → Taps "Request Ride" 🔵               ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑥  🖥️ Server creates ride → Status: "LOOKING FOR DRIVER"      ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑦  🚙 Nearby drivers get alert: "New Ride Available!"         ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑧  ✅ A driver taps "Accept" → Status: "DRIVER FOUND"         ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑨  📲 Rider sees: driver's name, car, location on map         ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑩  🚙 Driver drives to pickup → Rider watches on map          ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑪  📞 Rider & Driver can CHAT during the ride                 ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑫  🚙 Driver arrives → Taps "I'm Here"                        ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑬  🚗 Driver starts trip → Drives to destination               ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑭  🏁 Driver ends trip → Status: "COMPLETED"                  ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑮  💳 Payment happens (Wallet/Cash/Card)                      ●               │
│       ●      ↓                                                          ●               │
│       ●  ⑯  ⭐ Both rate each other 1-5 stars                          ●               │
│       ●      ↓                                                          ●               │

┌──────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                      │
│   🟩  Login Check     🟩  Ride Matching   🟩  Payment Math     🟩  Save Messages    │
│   🟩  Send Push 📲    🟩  Send Email 📧   🟩  Find Drivers     🟩  Track Cars 🚙   │
│                                                                                      │
│   What happens here:                                                                │
│   • Checks: "Is the password correct?"                                              │
│   • Decides: "Which driver is closest?"                                             │

│   • Watches: "Did the driver move? Update the rider!"                              │
│   • Auto-cancels: "No driver found in 2 minutes? Cancel."                          │
│   • Auto-offline: "Driver hasn't moved in 10 min? Go offline."                     │

SECTION 5: 🤖 THE AUTOMATIC STUFF (Things That Happen Without Humans)

┌──────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                      │
│   ⏰ Every 10 seconds:                                                              │
│   → Server checks: "Is anyone waiting for a driver?"                                │
│   → If 60 seconds passed → Search wider area + alert rider                         │
│   → If 120 seconds passed → Cancel ride: "No drivers nearby"                       │
│                                                                                      │
│   ⏰ Every 60 seconds:                                                              │
│   → Server checks: "Are drivers still active?"                                      │
│   → If no movement for 10 minutes → Auto-set driver to "Offline"                   │
│                                                                                      │
│   ⏰ Every night at 2 AM:                                                           │
│   → Server cleans old logs (older than 2 months)                                    │
│   → Except rides marked as "Keep Forever"                                           │
│                                                                                      │
└──────────────────────────────────────────────────────────────────────────────────────┘
